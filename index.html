Tables in HTML

  • HTML tables allow web developers to arrange data into rows and columns.
  • A table in HTML consists of table cells inside rows and columns.

HTML Tables Syntax

  • <table></table>
  • Each table row starts with a <tr> and ends with a </tr> tag.
    • tr stands for table row.
  • Each table cell is defined by a <td> and a </td> tag.
    • td stands for table data.
  • Sometimes you want your cells to be table header cells. In those cases use the <th> tag instead of the <td> tag

HTML Div Element

  • The <div> element defines a section in a document (block-level).
  • The <div> element is used as a container for other HTML elements.
  • The <div> element is by default a block element.
  • The <div> element has no required attributes.
  • The <div> element is often used to group sections of a web page together.

HTML Semantic Elements

  • Semantic elements = elements with a meaning.
  • A semantic element clearly describes its meaning to both the browser and the developer.
  • In HTML there are some semantic elements that can be used to define different parts of a web page:
    • HTML <section> Element :
      • The <section> element defines a section in a document.
      • According to W3C's HTML documentation: "A section is a thematic grouping of content, typically with a heading."
      • Examples of where a <section> element can be used: Chapters, Introduction, News items, Contact information.
    • HTML <article> Element :
      • The <article> element specifies independent, self-contained content.
      • Examples of where the <article> element can be used: Forum posts, Blog posts, User comments, Product cards, Newspaper articles.
    • HTML <header> Element :
      • The <header> element represents a container for introductory content or a set of navigational links.
      • A <header> element typically contains: one or more heading elements, logo or icon, authorship information.
    • HTML <footer> Element :
      • The <footer> element defines a footer for a document or section.
      • A <footer> element typically contains: authorship information, copyright information, contact information sitemap, back to top links.
    • HTML <nav> Element :
      • The <nav> element defines a set of navigation links.
    • HTML <aside> Element :
      • The <aside> element defines some content aside from the content it is placed in (like a sidebar).
      • The <aside> content should be indirectly related to the surrounding content.

HTML Audio & Video

  • HTML Audio

    • The HTML <audio> element is used to play an audio file on a web page.
    • The controls attribute adds audio controls, like play, pause, and volume.
    • The <source> element allows you to specify alternative audio files which the browser may choose from.
    • The text between the <audio> and </audio> tags will only be displayed in browsers that do not support the <audio> element.
    • To start an audio file automatically, use the autoplay attribute.
    • Chromium browsers do not allow autoplay in most cases. However, muted autoplay is always allowed.

  • HTML Video

    • The HTML <video> element is used to show a video on a web page.
    • The controls attribute adds video controls, like play, pause, and volume.
    • It is a good idea to always include width and height attributes.
    • The <source> element allows you to specify alternative video files which the browser may choose from.
    • The text between the <video> and </video> tags will only be displayed in browsers that do not support the <video> element.
    • To start an video file automatically, use the autoplay attribute.
    • Chromium browsers do not allow autoplay in most cases. However, muted autoplay is always allowed.

HTML Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

Example Code:

<form action="/submit" method="post">
<label for="userName">Username:</label>
<input type="text" id="userName" name="userName">
<label for="password">Password:</label>
<input type="password" id="password" name="password">
<input type="submit" value="Submit">

HTML Form Elements

Element Description
<form> Defines an HTML form for user input
<label> Defines a label for an input element
<input> Defines a form input element
<textarea> Defines a multi-line text input element
<select> Defines a drop-down list of options
<button> Defines a clickable button
<fieldset> Groups related elements in a form
<legend> Defines a caption for a fieldset
<option> Defines an option in a select element
<optgroup> Defines a group of options in a select element
<datalist> Defines a list of pre-defined options for an input element
<output> Defines the result of a calculation

Accessability in HTML

Accessability in HTML is used to make the website accessible to all users.

There are several ways to make a website accessible:

  • Use alt attribute for images
  • Use title attribute for links
  • Use label attribute for form elements
  • Use aria attribute for accessibility
